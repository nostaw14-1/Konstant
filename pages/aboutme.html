About the Author

Hey, I'm George, and this is my IST website project. I grew up in Sydney, Australia and have lived there my whole life. I've always had a passion for electronics and coding. My very first experience with coding was actually with HTML and CSS when I was around 9. The website I made was a verybasic website where I embedded some online games to a basic index page, with a rudimentary navigation system. I also have an interest in hardware and electronics. I've built many projects using raspberry Pi's. For example, I built a temperature sensor for the fridge at my dad's pharmacy, which logs the temperature and sends alerts when the temperature is over a certain threshold. I also built the computer that I still use when I was 11. I've continued to expand my expertice and understanding with the topic, refining code that I've written and upgrading the computer over the years, with this website being the most polished work I have made.
